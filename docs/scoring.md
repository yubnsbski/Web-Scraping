# 投資スコアリング機能

## 目的

投資スコアリング機能は、ユーザーが用意したローカルCSVをもとに、複数の投資候補を同じルールで機械的に比較するための補助機能です。

この機能は投資助言、売買推奨、将来リターンの保証ではありません。最終的な投資判断はユーザー本人が行います。自動売買は行いません。

## 特徴

- Gemini APIを呼びません。
- 外部マーケットデータを取得しません。
- ローカルCSVだけを入力として使います。
- 透明なルールでスコアを計算します。
- 自動売買や実注文は行いません。

## CSV形式

スコアリング対象のCSVには、次のカラムが必要です。

| カラム名 | 意味 | 評価方向 |
| --- | --- | --- |
| `name` | 投資候補名 | 表示用 |
| `expense_ratio` | 経費率 | 低いほど高評価 |
| `annual_return` | 年率リターン | 高いほど高評価 |
| `volatility` | ボラティリティ | 低いほど高評価 |
| `diversification_score` | 分散度スコア | 高いほど高評価 |

`diversification_score` は 0〜1 の値を指定してください。

## 入力例

```csv
name,expense_ratio,annual_return,volatility,diversification_score
