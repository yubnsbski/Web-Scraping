# アンサンブル予測（Phase 5）

実際の財務データ（Robert Shiller の長期 S&P 500 月次系列、GitHub `datasets/s-and-p-500`
ミラー）を用いて、複数のベース予測器を組み合わせたアンサンブル予測を**取得・評価・予測**まで
一貫して行います。これは教育・調査目的の統計的推定であり、**投資助言ではなく、自動売買も
行いません。最終的な投資判断はユーザー本人が行ってください。**

## 構成

- **ベース予測器（標準ライブラリのみ・依存なし）**
  - `naive`（持続＝ランダムウォーク基準）/ `drift` / `linear_trend`（OLS）/
    `holt_linear`（二重指数平滑）/ `ar{p}`（自己回帰・OLS正規方程式）
- **オプションML（`pip install -e '.[forecast]'` で numpy / scikit-learn）**
  - `random_forest`（バギング）/ `gradient_boosting`（ブースティング）
  - 未インストールでもコアは動作（遅延import）
- **アンサンブル結合**: `mean` / `median` / `weighted`（検証区間RMSEの逆数のべき乗で重み付け）
- **リターン空間オプション**: `--space returns` で各メンバーを対数リターンで学習し水準へ復元
  （`naive` 基準は水準のまま）
- **評価**: ウォークフォワード（拡大窓・将来データ非漏洩）で MAE / RMSE / MAPE /
  方向的中率 / naive 比スキルスコアを算出

## 使い方

```bash
# 1) 実データ取得（GitHubホスト。SSRF検証＋サイズ上限付き）
investment-assistant forecast-fetch-data --dest market/sp500.csv

# 2) 評価（リターン空間・加重アンサンブル、MLも含める場合は --no-ml を外す）
investment-assistant forecast-evaluate --path market/sp500.csv --tail 240 --space returns

# 3) 次ステップ予測（アンサンブル）
investment-assistant forecast-predict --path market/sp500.csv --horizon 1
```

ネットワークが使えない環境では、リポジトリ同梱の実データ小サンプル
`examples/sp500_monthly_sample.csv`（直近120ヶ月）で同じ評価・予測を再現できます。

## 評価で得られた主要な知見

直近240ヶ月・1ヶ月先予測・ML込みでのウォークフォワード結果（skill は naive 比、正で勝ち）:

| 空間 | 最良単独 | アンサンブル(weighted) | random_forest | linear_trend |
|------|----------|------------------------|---------------|--------------|
| level（価格水準を直接予測） | ar3 `+0.100` | `-0.263` | `-0.624` | `-5.866` |
| returns（対数リターンで予測） | ar3 `+0.116` | `+0.097` | `-0.005` | `+0.084` |

1. **株価水準はランダムウォークに近い**。naive（前月値の踏襲）は非常に強い基準で、
   多くのモデルのスキルは0付近。
2. **水準を直接予測する枠組みは不適切**。`linear_trend` は破綻（skill -5.9）、
   木モデル（RF/GBM）は学習レンジ外を外挿できず最高値更新時に系統的に過小予測
   （時系列MLの典型的な落とし穴、skill -0.62/-0.66）。
3. **対数リターン空間に変えると劇的に改善**。アンサンブルは `-0.263 → +0.097` と
   naive を上回り、単独最良の AR に肉薄。`linear_trend` は `-5.87 → +0.08` に回復。
4. **アンサンブルの劣悪メンバー耐性**。逆誤差重みを2乗（`weight_power=2`）にすることで、
   破綻メンバーの重みをほぼ0に抑え、結合の汚染を防止。
5. **実用的な既定値は「リターン空間 × 加重アンサンブル」**。事前にどのモデルが勝つか
   分からなくても naive を安定的に上回るのがアンサンブルの価値。本マクロ月次系列では
   依然 AR が強く、木モデルはこのデータ量・特徴量では優位を示さなかった。

## 注意・限界

- 月次マクロ系列での評価。日次・銘柄レベルや特徴量追加では結論は変わり得ます。
- スキルスコアは小さく、**「予測可能性は限定的」という事実そのものが重要な評価結果**です。
- 取得データには将来の不確実性があり、過去の当てはまりは将来の成績を保証しません。
